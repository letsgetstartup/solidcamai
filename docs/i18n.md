# Internationalization (i18n)

The Cloud Portal supports English and Hebrew.

## Local Files
- `ui/locales/en.json`: English strings.
- `ui/locales/he.json`: Hebrew strings with `__direction: "rtl"`.

## RTL Support
The UI automatically detects the locale and applies `dir="rtl"` to the wrapper, enabling correct Right-to-Left rendering for Hebrew.
